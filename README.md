\# Ntando's News



Ntando's News is a Django-based capstone news application that allows readers, journalists, editors, and publishers to interact in a structured digital newsroom.



\---



\# Features



\- Custom user model

\- Role-based access control

\- Reader, Journalist, and Editor roles

\- Publisher management

\- Article creation and approval workflow

\- Newsletter model

\- Reader subscriptions to publishers and journalists

\- REST API using Django REST Framework

\- JWT authentication

\- Automated API tests

\- Email notification logic

\- X/Twitter posting logic

\- Responsive front-end interface



\---



\# User Roles



| Role | Permissions |

|---|---|

| Reader | View articles and newsletters |

| Journalist | Create, view, update, and delete articles/newsletters |

| Editor | Review, approve, update, and delete articles/newsletters |
